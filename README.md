# 💬 D'Avilla Chat

Projeto de chat em tempo real utilizando WebSockets, com modo escuro, painel de usuários online, avatares personalizados e painel lateral de atualizações.

## 📦 Tecnologias utilizadas

- **Node.js** + **Express** – Servidor HTTP
- **WebSocket (ws)** – Comunicação em tempo real
- **HTML/CSS/JavaScript** – Interface do usuário
- **DiceBear Avatars API** – Avatares gerados automaticamente
- **Modo escuro com `localStorage`** – Persistência de tema

## 🚀 Como rodar o projeto

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/WuallanDAvilla/chat.git
   cd chat
   ```

2. **Instale as dependências:**

   ```bash
   npm install
   ```

3. **Inicie o servidor:**

   ```bash
   npm start
   ```

4. **Acesse no navegador:**

   ```
   http://localhost:3000
   ```

## 🌟 Funcionalidades

- ✅ Interface centralizada e responsiva
- 🌙 Modo escuro com persistência
- 👥 Lista de usuários online com avatares únicos
- 💬 Chat em tempo real com WebSocket
- 📜 Painel lateral com changelog
- 🧠 Cores e imagens de usuário geradas dinamicamente
